HOW TO RUN
1. Extract the ZIP.
2. Open the extracted folder.
3. Open CMD/Terminal in that folder.
4. Run: npm install
5. Run: npm run dev
6. Open the exact localhost address shown in the terminal.

Note: package.json is directly inside the extracted folder.
